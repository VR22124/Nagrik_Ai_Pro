# Google Services Architecture (Gemini Flash)

While NagrikAI Pro leverages Google Gemini, it utilizes AI specifically as a strictly bounded "Enhancement Layer", rather than an "Informational Generator". Generative output is unsafe for legal/compliance mapping natively; AI here operates only as a translator.

## Explain Mode (JSON -> English)
When activated, the AI receives the fully computed, factually correct JSON array generated by our deterministic rules engine. 
**Gemini Task Constraint**: "Rewrite the following fixed guidance into simple language. Do not add new facts." 
**Result**: Near-zero probability of hallucinating new legal routes. 

## Context-Aware Chat
The Chat assistant isn't generic. Every chat prompt silently ships the user's evaluated context parameter logic (e.g., `{ age: 22, guidanceState: "Form 8 Update" }`) underneath the message payload. Gemini aligns its responses strictly checking against this local rule snapshot. 

## Non-Political Mandate
The AI models are mapped alongside static system prompts enforcing strict non-partisanship:
* *Do not give political opinions, endorsements, or party comparisons.*
* *Follow Election Commission of India process language.*
These safety parameters prevent edge-case chat hijacking scenarios seamlessly.
